💡 Comment lire les pastilles couleur (eau / gaz / EDF)
🟢 Vert : dans le budget (bien)  ·  🟠 Orange : un peu au-dessus du plafond (+10 à +20 %)  ·  🔴 Rouge : nettement au-dessus du plafond (plus de +20 %)
Ces couleurs comparent le coût/kg de chaque jour à un plafond issu de ton budget (budget annuel du fluide ÷ tonnage annuel). 🟢 vert = ce jour est dans le rythme du budget ; 🟠/🔴 = au-dessus du plafond, donc trop cher au kg. Le repère est fixe : il alerte dès le 1ᵉʳ jour saisi, même si le mois est encore peu rempli.
